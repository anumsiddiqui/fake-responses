# Fake Responses

Generates .csv files of varying sizes, each filled with fake response data.

## Prerequisities

Install ruby and bundler.

Install gem dependencies:

```` sh
cd resources/data-analysis/fake-responses/
bundle install
````

## Usage

Generate data.

```` sh
ruby fake_responses.rb
````
